Uma aplicação desenvolvida em Python integrada ao PostgreSQL, com uma interface web para facilitar o gerenciamento de gastos e custos do dia a dia.

O sistema permite registrar despesas de forma organizada, categorizando cada gasto e oferecendo uma visão clara das finanças. Ideal para uso pessoal ou familiar, ajudando no controle financeiro e na tomada de decisões mais conscientes.

---

### Funcionalidades

- Registro de gastos e despesas

- Organização por categorias (alimentação, contas, transporte, etc.)

- Associação de gastos a pessoas (ex: membros da família)

- Visualização de histórico financeiro

- Interface web simples e intuitiva

- Integração com banco de dados PostgreSQL
